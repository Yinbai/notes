# Git

### 分支

1. 创建分支

   ``` bash
   git branch (branchname)
   ```

2. 切换分支

   ``` bash
   git checkout (branchname)
   git checkout -b (branchname)		// 创建并切换到分支
   ```

3. 查看所有分支

   ``` bash
   git branch			// 列出本地分支
   git branch -a		 // 列出所有分支
   
   
   ```

4. 删除分支

   ``` bash
   git branch -d (branchname)		// 删除分支
   ```
